Dear All,

In this e-mail we would like to update you on the progress of the tool
for zcash blockchain analysis platform (GrantProposals-2018Q2, grant
proposal #39).

Firstly, we established a dedicated repository for the tool (called
"Dru"): https://github.com/bergplace/Dru.

Secondly, we made a decision that Dru will consist from two layers:
- block-engine devoted for obtaining and block storage in a database
- frontend that will be responsible for interaction with the user (API,
handling requests, messaging)

This will allow to obtain and run only a docker image of the
block-engine for those who have different ideas for the use of the
system, not related to the frontent/API. Of course, in the grant project
we will deliver both layers.

We successfuly moved and abstracted the blockchain layer and as for now
the block-engine is capable of working with zcash.

We also await the security recommendations of the security expert who
joined and evaluates the project idea.

Apart from that we started to work on the documentation of the project.

Greetings,
Radoslaw Michalski & team
